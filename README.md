# Face SDK (Gawrs Face)

Face Recognition SDK for Server, Mobile or Embedded Solutions.

## Face SDK Features

- [ ] Face detection
- [ ] Face verification (1:1)
- [ ] Face identification (1:N)
- [ ] Face tracking

## Supported Operating Systems

- [ ] Windows x86 (32-bit, 64-bit)
- [ ] Linux x86 (64-bit)
- [ ] Linux ARM (64-bit)
- [ ] Android ARM (32-bit, 64-bit)
- [ ] iOS ARM (64-bit)

## Available APIs

- [ ] C++ API (for Windows, Linux, Android, iOS)
- [ ] Java wrapper (for Windows, Linux, Android)
- [ ] Python wrapper (for Windows, Linux)
- [ ] Swift API (for iOS)
- [ ] Flutter plugin (for iOS, Android)
